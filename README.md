# NovelAI キャラクタープロンプト図鑑

画像生成AI（NovelAIなど）で頻繁に使用するキャラクターを管理し、ワンクリックで簡単にクリップボードへコピーできるWebアプリケーションです。
コピーされる内容は、AIが最も認識しやすい `キャラクターの英名 (作品の英名)` の形式（Danbooruタグ形式）に最適化されています。

## 特徴

- **ワンクリックコピー**: キャラクターのカードをクリックするだけで、設定されたプロンプトがすぐにクリップボードにコピーされます。
- **画面上からの簡単操作**: 画面右下の「＋」ボタンから新しいキャラクターを追加したり、各カードの「×」ボタンから個別に削除することができます。
- **アニメDBからの自動取得**: 作品名を入力して「🪄 アニメDBから一括登録」ボタンを押すだけで、外部のデータベース（AniList API）からキャラクターの画像や英語名を自動で取得し、一括登録します。
- **ブラウザへの自動保存**: 画面から追加・削除したデータはブラウザ（ローカルストレージ）に自動保存されるため、次回開いたときもそのまま利用できます。
- **モダンなデザイン**: AIツールと親和性の高い、洗練されたダークテーマとアニメーションを採用しています。
- **環境構築不要**: HTML/CSS/JavaScriptのみで作成されており、ビルドやサーバー構築不要で、ブラウザで開くだけですぐに使用できます。

## 使い方

1. リポジトリをダウンロード（またはクローン）し、フォルダを展開します。
2. フォルダ内にある `index.html` をお好みのWebブラウザ（Google Chrome, Microsoft Edge など）で開きます。
3. 画面右下の「＋」ボタンから、キャラクターを登録します（手動登録、またはアニメDBからの自動取得が選べます）。
4. 画面に表示されたキャラクターのカードをクリックすると、「プロンプトをコピーしました！」という通知が表示され、クリップボードにコピーされます。
5. NovelAIなどの画像生成AIのプロンプト入力欄に貼り付けて使用してください。

## 注意事項

- **クリップボードへのコピー機能について**
  ブラウザのセキュリティ制限により、クリップボードAPI（`navigator.clipboard.writeText`）は以下の環境でのみ動作します。
  - ローカルファイルとして開いた場合（URLが `file://` から始まる状態）
  - セキュアな通信環境（`https://` または `http://localhost`）
  Webサーバーにアップロードして公開する場合は、必ずHTTPS接続が有効な環境をご利用ください。
- **データの保存場所について**
  画面から登録したデータは「現在開いているブラウザのローカルストレージ」に保存されます。別のブラウザや別のPCで開いた場合は、データは同期されませんのでご注意ください。

## ファイル構成
- `index.html` : メインのHTMLファイル
- `style.css` : デザインを定義するスタイルシート
- `main.js` : アプリケーションの動作（コピー機能や画面生成）を制御するプログラム
- `data.js` : 作品・キャラクター・プロンプトのデータを管理するファイル
