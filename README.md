# NovelAI キャラクタープロンプト図鑑

画像生成AI（NovelAIなど）で頻繁に使用する「キャラクターのプロンプト（呪文）」を管理し、ワンクリックで簡単にクリップボードへコピーできるWebアプリケーションです。

## ✨ 特徴

- **ワンクリックコピー**: キャラクターのカードをクリックするだけで、設定されたプロンプトがすぐにクリップボードにコピーされます。
- **モダンなデザイン**: AIツールと親和性の高い、洗練されたダークテーマとアニメーションを採用しています。
- **簡単なデータ管理**: データを別ファイル（`data.js`）に分離しているため、HTMLやCSSの知識がなくても簡単にキャラクターの追加や編集が可能です。
- **環境構築不要**: HTML/CSS/JavaScriptのみで作成されており、ビルドやサーバー構築不要で、ブラウザで開くだけですぐに使用できます。

## 🚀 使い方

1. リポジトリをダウンロード（またはクローン）し、フォルダを展開します。
2. フォルダ内にある `index.html` をお好みのWebブラウザ（Google Chrome, Microsoft Edge など）で開きます。
3. 画面に表示されたキャラクターのカードをクリックすると、「プロンプトをコピーしました！」という通知が表示され、プロンプトがコピーされます。
4. NovelAIなどの画像生成AIのプロンプト入力欄に貼り付けて使用してください。

## 📝 キャラクター（データ）の追加方法

キャラクターの追加・変更は、`data.js` ファイルをテキストエディタで編集するだけで行えます。

```javascript
// data.js の編集例
const worksData = [
    {
        title: "新しい作品名",
        characters: [
            {
                name: "キャラクター名",
                image: "画像のURLやローカルパス (例: ./images/char1.png)",
                prompt: "1girl, solo, ... (ここにプロンプトを記述)"
            }
        ]
    }
];
```

## ⚠️ 注意事項

- **クリップボードへのコピー機能について**
  ブラウザのセキュリティ制限により、クリップボードAPI（`navigator.clipboard.writeText`）は以下の環境でのみ動作します。
  - ローカルファイルとして開いた場合（URLが `file://` から始まる状態）
  - セキュアな通信環境（`https://` または `http://localhost`）
  Webサーバーにアップロードして公開する場合は、必ずHTTPS接続が有効な環境をご利用ください。

## 📁 ファイル構成
- `index.html` : メインのHTMLファイル
- `style.css` : デザインを定義するスタイルシート
- `main.js` : アプリケーションの動作（コピー機能や画面生成）を制御するプログラム
- `data.js` : 作品・キャラクター・プロンプトのデータを管理するファイル
